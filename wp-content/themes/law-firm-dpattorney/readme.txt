law-firm-dpattorney - Modern Law Firm WordPress Theme
====================================================

Author: AI Development Agent
Version: 1.0.0
Text Domain: law-firm-dpattorney

Description:
A modern, accessible, and authoritative law firm theme for WordPress. Built for Gutenberg, WPML/Polylang, and WCAG 2.1 AA compliance. Includes custom post types, customizer, and custom blocks.

Installation:
1. Upload the law-firm-dpattorney folder to wp-content/themes/
2. Activate via Appearance > Themes
3. Install recommended plugins: ACF/Carbon Fields (optional), WPML/Polylang (recommended)
4. Import demo-content.xml via Tools > Import
