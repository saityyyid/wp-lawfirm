# law-firm-dpattorney Theme Documentation

## Overview
A fully customizable, professional WordPress theme for law firms. Inspired by umbra.law and ahp.id.

## Features
- Customizer: All visual elements editable
- Gutenberg block support
- Custom post types: Attorneys, Practice Areas, Case Studies, Testimonials, FAQs
- Accessibility, SEO, performance optimized
- Preset color schemes

## Installation
1. Upload the theme folder to `/wp-content/themes/`.
2. Activate via WordPress admin.
3. Customize via Appearance > Customizer.
